◆ loop()

void loop ( )

This function checks for the incoming messages from Trackle, and processes any messages that have come in. It also sends keep-alive pings to the Cloud, so if it's not called frequently, the connection to Trackle may be lost. It also handles health check messages communication and keep up and running the connection to Trackle by making reconnections.
